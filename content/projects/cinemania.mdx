Cinemania is an interactive, frontend-heavy movie discovery engine allowing users to search, filter, and explore film data seamlessly.

### Overview
Built using **React** and bundled with **Vite**, the application prioritizes fast client-side navigation and immediate API response rendering. It interfaces with external movie databases to pull high-resolution posters, cast details, and real-time ratings.

### Performance & UI
The focus was on creating a "native application" feel within the browser. By aggressively optimizing component re-renders and handling image loading states efficiently, the grid layout scales beautifully from ultra-wide desktop monitors down to mobile screens.
